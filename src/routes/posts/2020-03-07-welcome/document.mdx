**1. 新建文章：** 在 `src/routes/posts`下按照 `yyyy-mm-dd-your-page-title`创建文件夹，将 markdown 添加到 `document.mdx` 中，再新建 `post.js` 添加文章信息：

```jsx
title: `文章标题`,
tags: ['文章标签1', '文章标签2'],
spoiler: "文章简略说明",
getContent: () => import('./document.mdx'),
```

**2. 编辑文章：** 比如要修改当前文章，找到`src/routes/posts/2020-03-07-welcome/document.mdx`，修改后保存，页面会自动热更新。

**3. 修改页面主题：** 编辑 `src/siteMetadata.js` metadata 属性。

**4. 编辑全局样式：** 在 `src/index.module.css` 修改 css 即可.

**5. 代码高亮：** 在对应的代码处添加花括号 {} 及对应行数，如 `jsx{1-4,6}`：

```jsx{1-4,6}
class Parent extends React.Component {
  render() {
    return (
      <MyFunctionalComponent ref={this.textInput} />
    );
  }
}

```

**7. 发布文章：** 文章编辑完毕，直接 git push 即可，注意需要在 master 分支上操作。
