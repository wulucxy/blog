# metadata api

## header

+ x-token 用户token
+ x-biz-token 系统token

## url

/metadata/{scope}/{model}/{?method}

## method

+ get 读取
+ post 创建
+ put 更新
+ delete 删除

## scope & model

索引字段可以作为查询条件，一次查询可以使用多个查询条件。

+ `scope`
+ project
    + `model`
  + list -- 项目列表 `get`
    + 联合查询 project_member & project_base （用户、用户组查项目）
  + info -- 项目信息 `get`
    + 联合查询 project_base & project_attach
  + base -- 项目基本信息 `get` `put` `post` `delete`
    + 查询表 project_base
  + member -- 项目成员 `get` `put` `post` `delete`
    + 查询表 project_member
  + attach -- 项目附加信息 `get` `put` `post` `delete`
    + 查询表 project_attach
  + log -- 项目信息日志 `get`
    + 查询表 project_log
+ group
  + `model`
  + group -- 项目组信息 `get` `put` `post` `delete`
    + 查询表 project_group
  + member -- 项目组成员 `get` `put` `post` `delete`
    + 查询表 project_group_member
  + project -- 项目组项目 `get` `put` `post` `delete`
    + 查询表 group_project
+ variable
  + `model`
  + variable -- 变量信息 `get`
    + 查询表 variable
+ team
    + `model`
  + list -- 团队列表 `get`
        + 联合查询 role_base & team_group_user (角色 / 用户列表)
  + base -- 团队信息 `get` `put` `post` `delete`
        联合查询 role_base & team_group_user (角色 / 用户列表)
  + roles -- 获取角色列表 `get` `put` `post` `delete`
        + 查询表 role_base
  + users -- 用户列表 `get` `put` `post` `delete`
        + 联合查询 role_base & team_group_user (角色 / 用户列表)
+ role
  + `model`
  + list -- 角色列表 `get`
    + 查询表 system_group_role
  + base -- 角色信息 `get` `put` `post` `delete`
    + 查询表 system_group_role
+ system
  + `model`
  + list -- 系统列表 `get`
    + 查询表 role_base 
  + roles -- 获取系统角色列表 `get` `put` `post` `delete`
    + 查询表 role_base
    + base -- 系统信息 `get` `put` `post` `delete`
    + 查询表 role_base
+ user
  + `model`
    + base -- 用户信息 `get` `put` `post` `delete`
    + 查询表 team_base project_member

## 示例

+ 查询某个项目的基本信息
  + /metadata/project/base?id=3
+ 查询项目组下的管理员
  + /metadata/group/member?group_id=3&role_id=
